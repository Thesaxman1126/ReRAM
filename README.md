# ReRAM
Code and data related to the ReRAM 

# SET UP
Using the terminal first clone the repo to the location of your choice by doing the following:
```
git clone https://github.com/Thesaxman1126/ReRAM.git
```

Once cloned make a virtual environment and activate it using the following commands
```
python3 -m venv venv
source venv/bin/activate
```

From here you can install all the required packages by using 
```
pip install -r requirements.txt
```
